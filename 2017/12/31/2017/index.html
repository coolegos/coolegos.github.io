2017

发表于 2017-12-31 | 分类于 总结

2017是工作的第三个年头了。

这一年有提升。
接触的项目更加的深入，对项目时间的评估、规划等等也比较的熟练了。
Android 知识在这一年有提升，最主要的是开始去关注Android Framework(开始去习惯Android 源码，觉得领悟的比较晚)。
开始去认真的思考和学习计算机方面知识(这一年思考过大学毕业以后对计算机的理解，总感觉欠缺很多)，主要是网络、操作系统和一些算法。
开始去思考自己的职业规划(觉得遇到了瓶颈)，主要是去寻找自己的定位(目前还没有完全定位，但是有花时间在想这些事情)。

这一年打算做但最终没有坚持的事情有三件，react native、个人博客和学习AI。
react native 做了大概一个月，主要是集成进现有的工程。
个人博客在年初势头很大，后期发现自己写的内容质量太低。
学习AI 最终只做到了python 入门。
现在年尾再来回顾这三件事，每一件都是最开始势头很足，然后后面出现了瓶颈，最终不了了之了。

回过头来想给自己说上几句话(主要是自我批评的话)。
1.要多交流，去了解不足，不能一直把自己定位在熟悉的方向。
2.真的要坚持，一件事情总是能碰到瓶颈，只有当跳过了一个瓶颈以后才能看到更广阔的天空，不然一直是井底之蛙。
3.要打开自己的眼界，善待他人，善待生活。

以往的每一年都会在印象笔记记录总结，但总是太过于主观以及慷慨激让(总是太多计划并且很矫情，所以没有公开在博客)。这一次尽量写出来的比较的客观，去分析一些自己碰到的问题。

2018的打算本来也想很具体的写写，但是每次一年的打算总是没有很好的实现，所有就不一一列举一定要做到什么。对2018 给个自己的关键词，坚持。
